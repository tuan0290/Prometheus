# Lab 06 - Service Discovery

## Tổng Quan

Lab này hướng dẫn bạn thiết lập service discovery để tự động discover targets. Bạn sẽ configure file-based service discovery hoặc Kubernetes service discovery.

**Cấp độ**: Advanced  
**Thời gian ước tính**: 120 phút  
**Yêu cầu trước**: Hoàn thành Lab 05 và Components module

## Mục Tiêu

Trong bài lab này, bạn sẽ:
- Configure file-based service discovery
- Setup relabeling rules
- (Optional) Configure Kubernetes service discovery
- Verify dynamic target discovery
- Understand service discovery mechanisms
