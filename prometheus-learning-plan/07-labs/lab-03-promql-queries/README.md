# Lab 03 - PromQL Queries

## Tổng Quan

Lab này cung cấp bài tập thực hành để làm chủ PromQL. Bạn sẽ viết queries từ cơ bản đến nâng cao để truy vấn và phân tích metrics.

**Cấp độ**: Intermediate  
**Thời gian ước tính**: 120 phút  
**Yêu cầu trước**: Hoàn thành Lab 02 và PromQL module

## Mục Tiêu

Trong bài lab này, bạn sẽ:
- Viết instant vector và range vector queries
- Sử dụng operators để kết hợp metrics
- Áp dụng functions như rate(), increase(), sum()
- Thực hiện aggregations với grouping
- Viết complex queries với subqueries
