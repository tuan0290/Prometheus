# Lab 04 - Alertmanager

## Tổng Quan

Lab này hướng dẫn bạn cấu hình alerting với Alertmanager. Bạn sẽ tạo alerting rules, configure routing, và thiết lập notification channels.

**Cấp độ**: Advanced  
**Thời gian ước tính**: 120 phút  
**Yêu cầu trước**: Hoàn thành Lab 03 và Alerting module

## Mục Tiêu

Trong bài lab này, bạn sẽ:
- Viết alerting rules với PromQL
- Cài đặt và cấu hình Alertmanager
- Thiết lập routing tree với grouping
- Configure notification channels (Email, Slack)
- Test alert firing và notification delivery
