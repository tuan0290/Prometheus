# Module 07: Labs

**Cấp độ**: All Levels  
**Thời gian ước tính**: 15 giờ  
**Yêu cầu trước**: Varies by lab

## Tổng Quan

Module này cung cấp bài tập thực hành hands-on để áp dụng kiến thức lý thuyết. Mỗi lab bao gồm hướng dẫn từng bước, expected outcomes, troubleshooting tips, và verification steps.

## Mục Tiêu Học Tập

Sau khi hoàn thành module này, bạn sẽ có thể:
- Cài đặt và cấu hình Prometheus từ đầu
- Tạo và expose custom metrics
- Viết và test PromQL queries
- Cấu hình Alertmanager và notifications
- Tích hợp multiple exporters
- Thiết lập service discovery
