# Module 03: PromQL

**Cấp độ**: Intermediate  
**Thời gian ước tính**: 10 giờ  
**Yêu cầu trước**: Hoàn thành Components

## Tổng Quan

Module này dạy ngôn ngữ truy vấn PromQL từ cơ bản đến nâng cao. Bạn sẽ học cách viết queries để truy vấn, phân tích, và visualize metrics data.

## Mục Tiêu Học Tập

Sau khi hoàn thành module này, bạn sẽ có thể:
- Viết PromQL queries với selectors và matchers
- Sử dụng operators (arithmetic, comparison, logical)
- Áp dụng built-in functions (rate, increase, sum, avg, etc.)
- Thực hiện aggregations với by và without
- Viết advanced queries với subqueries và recording rules

## Checkpoint

Đánh dấu khi bạn hoàn thành:
- [ ] Viết được instant vector và range vector selectors
- [ ] Sử dụng được operators để kết hợp metrics
- [ ] Áp dụng được rate() và increase() cho counters
- [ ] Thực hiện được aggregations với grouping
- [ ] Viết được complex queries với subqueries
