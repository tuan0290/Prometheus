# Module 01: Fundamentals

**Cấp độ**: Beginner  
**Thời gian ước tính**: 8 giờ  
**Yêu cầu trước**: Hoàn thành Getting Started

## Tổng Quan

Module này xây dựng nền tảng kiến thức cơ bản về monitoring, observability, và Prometheus. Bạn sẽ học về time-series databases, các loại metrics, kiến trúc Prometheus, và thuật ngữ chuyên ngành.

## Mục Tiêu Học Tập

Sau khi hoàn thành module này, bạn sẽ có thể:
- Phân biệt giữa monitoring và observability
- Hiểu cách time-series databases hoạt động
- Nắm vững 4 loại metrics: counter, gauge, histogram, summary
- Giải thích kiến trúc pull-based của Prometheus
- Sử dụng đúng thuật ngữ Prometheus trong giao tiếp

## Checkpoint

Đánh dấu khi bạn hoàn thành:
- [ ] Hiểu sự khác biệt giữa metrics, logs, và traces
- [ ] Có thể giải thích time-series data model
- [ ] Biết khi nào dùng counter, gauge, histogram, summary
- [ ] Hiểu kiến trúc pull-based và các thành phần chính
- [ ] Nắm vững các thuật ngữ Prometheus cơ bản
