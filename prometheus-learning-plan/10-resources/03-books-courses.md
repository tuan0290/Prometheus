# Books & Courses

## Books

### Prometheus: Up & Running
- **Tác giả**: Brian Brazil
- **Nhà xuất bản**: O'Reilly Media
- **Mô tả**: Cuốn sách chính thức về Prometheus, viết bởi core maintainer. Bao gồm từ cơ bản đến nâng cao.
- **Phù hợp**: Beginner đến Advanced
- **Link**: https://www.oreilly.com/library/view/prometheus-up/9781492034131/

### Site Reliability Engineering (Google SRE Book)
- **Tác giả**: Google SRE Team
- **Nhà xuất bản**: O'Reilly Media
- **Mô tả**: Nền tảng về SRE, monitoring, và observability. Free online.
- **Phù hợp**: Intermediate đến Advanced
- **Link**: https://sre.google/sre-book/table-of-contents/

### The Site Reliability Workbook
- **Tác giả**: Google SRE Team
- **Mô tả**: Practical guide với examples. Free online.
- **Link**: https://sre.google/workbook/table-of-contents/

### Cloud Native Monitoring with Prometheus
- **Tác giả**: Rob Skillington
- **Mô tả**: Prometheus trong cloud native environments
- **Phù hợp**: Intermediate

## Learning Paths

### Beginner Path (2-4 tuần)
1. Prometheus: Up & Running (Chapters 1-5)
2. Linux Foundation LFS241 course
3. Lab 01 - Installation
4. Lab 02 - Custom Metrics

### Intermediate Path (4-6 tuần)
1. Complete Prometheus: Up & Running
2. PromQL deep dive
3. Lab 03 - PromQL Queries
4. Lab 04 - Alertmanager

### Advanced Path (6-8 tuần)
1. SRE Book (monitoring chapters)
2. Thanos/Cortex for HA
3. Kubernetes monitoring
4. PCA certification prep
