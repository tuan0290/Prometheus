# Module 05: Alerting

**Cấp độ**: Advanced  
**Thời gian ước tính**: 8 giờ  
**Yêu cầu trước**: Hoàn thành Instrumentation

## Tổng Quan

Module này dạy cách cấu hình alerting và dashboarding. Bạn sẽ học về alerting rules, Alertmanager configuration, notification channels, và Grafana integration.

## Mục Tiêu Học Tập

Sau khi hoàn thành module này, bạn sẽ có thể:
- Viết alerting rules với PromQL
- Cấu hình Alertmanager routing, grouping, inhibition
- Thiết lập notification channels (Email, Slack, PagerDuty)
- Tạo Grafana dashboards và panels
- Quản lý alert lifecycle

## Checkpoint

Đánh dấu khi bạn hoàn thành:
- [ ] Viết được alerting rules với for và labels
- [ ] Cấu hình được Alertmanager routing tree
- [ ] Thiết lập được notification channels
- [ ] Tạo được Grafana dashboards
- [ ] Hiểu alert states: inactive, pending, firing
