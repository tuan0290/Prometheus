# Module 06: Integrations

**Cấp độ**: Advanced  
**Thời gian ước tính**: 10 giờ  
**Yêu cầu trước**: Hoàn thành Alerting

## Tổng Quan

Module này hướng dẫn tích hợp Prometheus với các platforms và tools phổ biến. Bạn sẽ học cách monitor Kubernetes, Docker, cloud providers, và xây dựng complete monitoring stacks.

## Mục Tiêu Học Tập

Sau khi hoàn thành module này, bạn sẽ có thể:
- Monitor Kubernetes clusters với Prometheus
- Thiết lập Docker container monitoring
- Tích hợp với AWS, GCP, Azure monitoring
- Xây dựng complete monitoring stack (Prometheus + Grafana + Alertmanager)
- Sử dụng kube-state-metrics và cAdvisor

## Checkpoint

Đánh dấu khi bạn hoàn thành:
- [ ] Cấu hình được Prometheus cho Kubernetes
- [ ] Monitor được Docker containers
- [ ] Tích hợp được cloud provider exporters
- [ ] Xây dựng được complete monitoring stack
