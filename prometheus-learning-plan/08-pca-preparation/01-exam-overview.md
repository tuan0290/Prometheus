# Prometheus Certified Associate (PCA) - Exam Overview

## Tổng Quan

Prometheus Certified Associate (PCA) là chứng chỉ do Linux Foundation cấp, xác nhận kiến thức về Prometheus monitoring system. Đây là chứng chỉ entry-level dành cho những người muốn chứng minh năng lực với Prometheus.

**Link đăng ký**: [https://training.linuxfoundation.org/certification/prometheus-certified-associate/](https://training.linuxfoundation.org/certification/prometheus-certified-associate/)

## Thông Tin Kỳ Thi

| Thông tin | Chi tiết |
|-----------|----------|
| Tên chứng chỉ | Prometheus Certified Associate (PCA) |
| Tổ chức cấp | Linux Foundation |
| Hình thức | Multiple choice, online proctored |
| Thời gian | 90 phút |
| Số câu hỏi | ~60 câu |
| Điểm đậu | 75% |
| Hiệu lực | 2 năm |
| Ngôn ngữ | Tiếng Anh |
| Chi phí | $250 USD |

## Chi Tiết Từng Domain

### Domain 1: Observability Concepts (18%)

Kiến thức về monitoring và observability:

- Metrics, Logs, Events, Traces
- Push vs Pull model
- Service Level Indicators (SLIs)
- Service Level Objectives (SLOs)
- Service Level Agreements (SLAs)
- Monitoring best practices

### Domain 2: Prometheus Fundamentals (20%)

Kiến thức cơ bản về Prometheus:

- Prometheus architecture
- Data model và time series
- Metric types (Counter, Gauge, Histogram, Summary)
- Scrape configuration
- Service discovery mechanisms
- Storage và retention
- Federation

### Domain 3: PromQL (28%)

Ngôn ngữ query của Prometheus (trọng số cao nhất):

- Selectors và matchers
- Data types (instant vector, range vector, scalar)
- Operators (arithmetic, comparison, logical)
- Functions (rate, increase, histogram_quantile, etc.)
- Aggregation operators
- Subqueries
- Recording rules

### Domain 4: Instrumentation and Exporters (20%)

Cách instrument applications và sử dụng exporters:

- Client libraries (Go, Python, Java, Ruby)
- Metric naming conventions
- Label best practices
- Common exporters (node, blackbox, mysql, postgres)
- Custom exporters
- Pushgateway

### Domain 5: Alerting and Dashboarding (14%)

Alerting và visualization:

- Alerting rules syntax
- Alert states (inactive, pending, firing)
- Alertmanager configuration
- Routing và grouping
- Notification channels
- Grafana integration
- Dashboard best practices

## Hình Thức Thi

### Online Proctored

- Thi tại nhà hoặc văn phòng
- Cần webcam và microphone
- Proctor giám sát qua video
- Không được dùng tài liệu (closed book)
- Cần ID hợp lệ

### Môi Trường Thi

- Browser-based exam platform
- Multiple choice questions
- Có thể flag câu hỏi để review
- Countdown timer hiển thị

## Yêu Cầu Kỹ Thuật

- Máy tính với webcam
- Internet connection ổn định
- Browser: Chrome hoặc Firefox mới nhất
- Tắt tất cả ứng dụng khác
- Phòng yên tĩnh, đủ ánh sáng

## Chuẩn Bị Trước Khi Thi

### 1 Tuần Trước

- [ ] Review tất cả 5 domains
- [ ] Làm practice questions
- [ ] Identify weak areas
- [ ] Review PromQL functions và operators

### 1 Ngày Trước

- [ ] Light review, không học quá nhiều
- [ ] Test technical setup (webcam, internet)
- [ ] Chuẩn bị ID
- [ ] Ngủ đủ giấc

### Ngày Thi

- [ ] Ăn sáng đầy đủ
- [ ] Vào phòng thi 30 phút trước
- [ ] Test webcam và microphone
- [ ] Dọn dẹp bàn làm việc

## Chiến Lược Làm Bài

1. **Đọc kỹ câu hỏi**: Chú ý từ khóa như "always", "never", "best practice"
2. **Loại trừ đáp án sai**: Thường có 1-2 đáp án rõ ràng sai
3. **Flag câu khó**: Làm câu dễ trước, quay lại câu khó sau
4. **Quản lý thời gian**: ~90 giây/câu
5. **Review**: Dành 10-15 phút cuối để review flagged questions
